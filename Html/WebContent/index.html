Block Level Elements

Default behaviour of block level element is

  • fill the container unless width is specifed
  • insert new line before and after contents
  • can not be nested within inline element
  • can contain inline elements
  • can contain other block Elements
  • Example : DIV , P-Paragraph, H1-H6- Header Tags, List tag and items

Inline Elements

Default behaviour of Inline element is
  • Goes with the flow
  • Cant have block elements
  • Can have other inline elements
  • Example : em-emphasis, strong,bold,italic
